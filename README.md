MONGO_URL=mongodb://dev:9fifLMdv1QjG1cqCgj6D@127.0.0.1:27017/oak-center?retryWrites=true&w=majority

## Launch

### Development

```
yarn && yarn run dev
```

### Production
```
yarn && yarn start
```
